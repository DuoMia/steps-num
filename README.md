# StepSync - 小米运动自动刷步数

一个简单易用的小米运动步数同步工具，支持可视化界面管理账号和配置。

## 功能特点

- **可视化界面** - 通过浏览器管理账号、配置步数
- **安全存储** - 配置保存在本地，不上传到任何服务器
- **多账号支持** - 支持多个小米运动账号
- **推送通知** - 支持 PushPlus、企业微信通知

## 安装运行

```bash
# 1. 安装依赖
pip install requests pytz pycryptodome flask

# 2. 启动服务
python webapp.py

# 3. 打开浏览器访问
# http://127.0.0.1:5000
```

### 配置说明

| 字段 | 说明 |
|------|------|
| 账号 | 小米运动账号（手机号或邮箱） |
| 密码 | 小米运动账号密码 |
| 最小步数 | 随机步数下限，默认 18000 |
| 最大步数 | 随机步数上限，默认 25000 |
| PushPlus Token | 微信推送通知（可选） |
| 企业微信 Key | 企业微信机器人通知（可选） |

### 多账号设置

在账号和密码字段中用 `#` 分隔多个账号：

- 账号: `13800138000#13800138001`
- 密码: `password1#password2`

### 步数规则

- 步数会根据当前时间自动调整
- 早上步数较少，晚上步数较多
- 北京时间 22 点达到设定的最大值

## 注意事项

1. 账号是 **小米运动/Zepp Life** 账号，不是小米账号
2. 小米运动不会更新步数，只有关联的第三方应用才会同步
3. 如果支付宝等没有更新步数，尝试注销后重新登录
4. 如刷步数失败，建议先去测试网站验证账号：
   - https://steps.hubp.de/
   - https://bs.yanwan.store/run4/
